⚡ Quick Start — 5 Minutes to Productivity

Type any of these in the Global Command Bar at the top of the screen, then press Enter or click GO:

AAPL US DES GO          → Apple description screen
MSFT US HP GO           → Microsoft historical prices
EURUSD Curncy GP GO     → EUR/USD daily price chart
WEI GO                  → World equity indices monitor
TOP GO                  → Top news headlines
GEO GO                  → Global intelligence map
NAVTREE GO              → Browse all 2,900+ functions
TUTOR GO                → This guided walkthrough

Command Grammar

<TICKER> <MARKET> <MNEMONIC> GO
PartExampleMeaning
TICKERAAPLThe symbol
MARKETUS / Curncy / Index / Corp / ComdtyAsset class suffix
MNEMONICDES / HP / GP / FAThe function to run
GOGOExecutes the command
TIMEFRAME5D / 1M / 3M / 1Y / 5YOptional — for charts/history
Typo correction: The command parser uses fuzzy matching. Typing WEL resolves to WEI. Close-enough always works.

🔍 Search — HL / Ctrl+K

Open with Ctrl+K, HL GO, or the HL button in the panel toolbar.

Searches across: Functions (2,949 mnemonics) · Securities · Fields · Monitors · Workspaces · News

Examples: Type div → finds DVD (Dividend History). Type ownership → finds OWN. Type options chain → finds CHAIN.

📊 Workflow: Build a Monitor

  1. MON GO → Open Monitor / Watchlist
  2. Type symbol + Enter → Add AAPL US Equity, MSFT US Equity, etc.
  3. MON+ GO → Advanced Monitor Builder (custom columns)
  4. FLD GO → Find fields → click Add→MON to add as column
  5. ALRT+ GO → Set field-based alerts for watchlist symbols
  6. WS mymonitor GO → Save as named workspace

🔔 Workflow: Set & Manage Alerts

  1. ALRT+ GO → Open Advanced Alerts
  2. Enter symbol (e.g. AAPL US Equity)
  3. Select field (e.g. PX_LAST), operator (> or <), threshold
  4. Click CREATE
  5. ALRT GO → View all rules + triggered status
Note: Alerts evaluate against simulated streaming data. System strip badge shows triggered count.

📋 Mnemonic Reference — Core Functions

WEI  World Equity Indices  🌍 Global  Market Monitors

Purpose: Real-time overview of all global equity indices. Shows level, change, %change, YTD, and region tags.

Open with: WEI GO

Related: IMAP · TOP · ECO · FXC · RFCM · GMOV

  • Click any index row → opens DES for that index
  • Column header click → sort
  • Shift+Click → opens in a new pane

DES  Security Description  🔒 Security Scoped  Equity Reference

Purpose: Comprehensive reference sheet: fundamentals (P/E, EPS, Div Yield, Market Cap, Beta), trading stats, business summary, peers, ownership, and events.

Open with: AAPL US DES GO

Related: HP · FA · OWN · CN · EVT · RELS · MGMT · DVD

  • Always include market suffix: AAPL US not just AAPL
  • Alt+Click any numeric value → Inspector with field lineage
  • F2 → MENU shows all related functions

HP  Historical Pricing  🔒 Security Scoped  Equity Reference

Purpose: Day-by-day OHLCV table. Sortable columns. Supports 1M, 3M, 1Y, 5Y timeframes.

Open with: MSFT US HP GO | MSFT US HP 1Y GO

Related: GP · GIP · DES · DVD

  • Add timeframe: AAPL US HP 1Y GO
  • PageDown → next page
  • Click column header → sort

GP  Price Chart  🔒 Security Scoped  Charts

Purpose: Canvas candlestick/line chart with volume bars, OHLC crosshair, and related ticker chips.

Open with: AAPL US GP GO | EURUSD Curncy GP 1Y GO

Related: GIP · HP · RV · DES

  • Hover over chart → shows OHLC values
  • Click ticker chip → opens GP for that ticker
  • Add timeframe: 5D, 1M, 3M, 1Y, 5Y

FA  Financial Analysis  🔒 Security Scoped  Equity Reference

Purpose: Three-tab financial statements viewer: Income Statement, Balance Sheet, Cash Flow. Multi-period columns.

Open with: NVDA US FA GO

Related: DES · HP · DVD

  • Click IS / BS / CF tabs to switch statements
  • Click any line item value → LINE lineage viewer

OWN  Institutional Ownership  🔒 Security Scoped  Equity Reference

Purpose: Ranked institutional holders with stake %, shares held, value, and recent changes.

Open with: MSFT US OWN GO

Related: DES · MGMT · RELS · RELG

  • Click holder row → CMPY dossier for that institution
  • Stake % and changes are SIM-sourced

TOP  Top News / News Hub  🌍 Global  News

Purpose: Global top news feed with headline tape, extracted entities, impacted tickers, and theme tags.

Open with: TOP GO | AAPL US CN GO

Related: CN · N · NMAP · NREL · NTIM · SENT

  • Click headline → NTIM (news timeline with price reaction)
  • Click ticker chip → DES
  • Right-click → context menu with "Open NREL"

ECO  Economic Calendar  🌍 Global  Macro

Purpose: Upcoming macro releases with country, importance, consensus, prior, and actual (post-release).

Open with: ECO GO

Related: WEI · TOP · CAL24 · FXC

  • Click country → CTY country dossier
  • Click event row → drill to the relevant macro function

FXC  FX Cross Matrix  🌍 Global  FX

Purpose: N×N matrix of spot FX rates for all major pairs.

Open with: FXC GO

Related: DES · GP · ECO · CORR+

  • Click any cell → DES for that FX pair
  • Right-click → GP chart options

GEO  Global Intelligence Map  🌍 Global  Geo Intelligence

Purpose: Interactive world map (MapLibre GL). Click any country to drill into region dossiers, news overlays, and supply chain signals.

Open with: GEO GO

Related: GEO.N · GEO.C · GEO.R · GEO.M · RGN · NMAP · SCN

  • Click country → RGN region dossier
  • Shift+Click → opens in new pane
  • Use GEO.N / GEO.C / GEO.R / GEO.M / GEO.S / GEO.E / GEO.F / GEO.A for overlays

MON  Monitor / Watchlist  🌍 Global  Monitors

Purpose: Streaming watchlist with custom columns from the Field Catalog. Multiple lists, sorting, filtering.

Open with: MON GO

Related: MON+ · FLD · ALRT+ · ALRT · WS

  • Type symbol + Enter to add
  • Use MON+ for advanced column builder
  • Symbols need full format: AAPL US Equity

ALRT  Alerts Monitor  🌍 Global  Monitors

Purpose: Shows all alert rules with triggered/inactive status. Drill into triggered alerts for evidence.

Open with: ALRT GO | ALRT+ GO

Related: ALRT+ · MON · MON+ · NOTIF

  • ALRT+ GO → create field-based threshold alerts
  • System strip top bar shows alert count badge
  • Alerts evaluate against simulated streaming data

RELG  Relationship Graph  🔒 Security Scoped  Relationships

Purpose: Visual relationship graph: peer / supply-chain / ownership / correlation links for any security.

Open with: AAPL US RELG GO

Related: RELT · EVID · PATH · SCN · CUST · OUT

  • Click node → DES of related company
  • Click edge → EVID evidence panel
  • Enter key → expand selected node

WS  Workspace Manager  🌍 Global  Platform

Purpose: Save and restore complete workspace states: pane layouts, panel content, command histories, pin strip.

Open with: WS GO | WS myname GO | WS DEL myname GO

Related: DOCK · FLOAT · LAYOUT · SNAP

  • WS myname GO → saves OR loads (uses existing if found)
  • WS:MARKET-WALL / NEWSROOM / RESEARCH / TRADING → load presets
  • Workspaces stored in localStorage — export before clearing browser

NAVTREE  Function Catalog  🌍 Global  Platform

Purpose: Browse, filter, and launch all 2,949+ functions. Category filters, taxonomy grouping, favorites/recent.

Open with: NAVTREE GO

Related: TUTOR · PREF · KEYMAP · NX

  • Type to filter instantly
  • Category dropdown → filter by asset class
  • Click any row → opens that function

FLD  Field Catalog  🌍 Global  Data & Lineage

Purpose: Searchable registry of all data fields with definitions, units, cadence, and asset class availability.

Open with: FLD GO

Related: LINE · MAP · QLT · MON+

  • Search by keyword, field ID, or definition
  • Click Add→MON to add as a monitor column
  • Click Lineage → opens LINE for that field

LINE  Data Lineage Viewer  🌍 Global  Data & Lineage

Purpose: Visual lineage trace: source → normalization transforms → displayed value. Shows SIM/LIVE/STALE provenance.

Open with: LINE GO | LINE PX_LAST GO

Related: FLD · MAP · QLT · SRC

  • Click any numeric value in DES/WEI/HP → auto-opens LINE
  • Alt+Click → Inspector shows field provenance in FIELDS section

PREF  Preferences & Settings  🌍 Global  Platform

Purpose: User settings: display density, layout mode, streaming mode, and advanced settings links.

Open with: PREF GO

Related: KEYMAP · GRIDCFG · THEMEPRO · FORMAT

  • Density changes persist across sessions via localStorage
  • Live Mode increases streaming rate for all panels

TUTOR  Guided Tutorial  🌍 Global  Platform

Purpose: 3-track guided walkthrough. Core Workflow, Global Map Stack, Platform & Admin. Click any row to open that function.

Open with: TUTOR GO | HELP GO

Related: NAVTREE · PREF · KEYMAP · DOCS

  • Use "Other Pane" mode to keep TUTOR open while exploring
  • HELP GO and TUTOR GO both open this screen
  • F1 inside any panel → contextual help for that panel

AUD  Command Audit Log  🌍 Global  Governance

Purpose: Chronological log of all user actions: GO commands, drills, alerts, exports, policy blocks.

Open with: AUD GO

Related: AUDIT · TRAIL · NAV · COMP · POL

  • Filter by type: GO, DRILL, ALERT_CREATE, EXPORT, POLICY_BLOCK
  • Export before clearing browser data — log persists in localStorage

IssueSolution
Panel is empty / shows home screenType any command + GO. E.g. WEI GO
Data shows SIM or STALEAll data is simulated by design. Enable Live Mode in PREF for more updates.
Can't access a function (policy block)Check COMP GO → compliance mode. Check POL GO → policy rules.
Drill not workingRight-click the row to see context menu. Alt+Click for Inspector overlay.
Monitor has no dataAdd symbols in MON: type AAPL US Equity + Enter. Full format required.

Order entry is blockedCheck KILL GO — kill switch may be active. Reset via COMP GO.
