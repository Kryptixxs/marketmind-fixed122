1. What Is This Terminal?

MarketMind Terminal is a professional-grade financial data platform modeled after Bloomberg Terminal. It is designed for analysts, traders, and investors who need fast, dense, keyboard-driven access to market data across multiple securities and asset classes simultaneously.

Key Concept
Instead of navigating between pages, you type short command codes called mnemonics (like DES, GP, WEI) and press Enter. The data loads directly into your focused panel. Everything is driven by the command line.

What makes it different from a normal web app

  • There are NO pages — you never click "navigate to" a URL. You type commands.
  • You have 4 independent panels open at once, each showing different data.
  • Everything you see is clickable: tickers, sectors, people, news headlines, fields.
  • All data is simulated (SIM mode) with realistic market behavior — no live exchange connection needed.
  • Price data updates automatically; cells flash green/red as prices change.

3. The 4-Panel System

The screen is always divided into four independent panels (P1–P4) arranged in a 2×2 grid. Each panel is completely independent — it has its own security, function, history, and command line.

Focusing a Panel

Only one panel is "focused" at a time — indicated by a bright blue border. Commands you type go to the focused panel only. Click any panel to focus it, or use keyboard shortcuts.

Click a panelFocus that panel
Alt+1 / 2 / 3 / 4Focus Panel 1, 2, 3, or 4 directly
Ctrl+` or Ctrl+TabCycle focus clockwise through all panels

Each panel stores independently:

  • Active security (e.g. AAPL US Equity)
  • Active function/mnemonic (e.g. DES)
  • Navigation history (up to 40 entries)
  • Timeframe for charts (5D / 1M / 1Y etc.)
  • Scroll position and selected row
  • Recent securities & favorites
  • Link group color (for panel sync)

Back / Forward Navigation:

Every panel tracks a history of where you've been — like a browser, but per panel.

  • Ctrl+B — Go back one step in this panel's history
  • Ctrl+Shift+B — Go forward
  • Click ◀ or ▶ in the toolbar
  • History restores scroll position and row selection exactly

4. The Command Line — How to Navigate

The amber command line at the bottom of each panel is your primary navigation tool. Type a command and press Enter (the "GO" key) to execute it.

Golden Rule
Type the security you want, then the function you want, then press Enter.
Example: AAPL US DES then Enter → loads Apple's description screen.

Autocomplete

As you type, a dropdown appears with matching securities (shown with →) and mnemonics/functions (shown with fn). Navigate with ↑↓, confirm with Enter or Tab, close with Esc.

Special Command Keys

Enter (GO)Execute the command
Esc (CANCEL)Clear input OR close any open overlay
F1Open help for the current function (press twice within 0.6s for Help Desk)
F2Open MENU — shows related functions for current security & sector
Ctrl+KOpen HL Search — search all securities and functions
Ctrl+LFocus the command line (from anywhere)

8. Clicking & Drilling Into Data

In MarketMind Terminal, everything visible is clickable. Tickers, index names, sectors, countries, analyst names, news headlines, data fields — all are linked to deeper data. The system knows what type of object it is and opens the right function automatically.

Three Ways to Click

Normal clickOpen in the current panel (replaces current view, pushes to history)
Shift + clickOpen in a new panel (sends to the next available panel; switches focus there)
Alt + clickOpen Inspector overlay — see dense field details WITHOUT leaving the current view
Right-clickContext menu with entity-specific actions: Open, Open in panel, Inspect, DES/GP/CN, Copy symbol, Add to monitor, Create alert
Middle-clickOpen Inspector overlay (same as Alt+click)

Smart Memory
The terminal remembers the last function you used for each security in each panel. If you previously looked at AAPL's chart (GP), then navigate away and click AAPL again — it opens GP, not DES. This is "last-used mnemonic" memory.

The Drill Chain — How Deep Can You Go?

WEI→ click index → DES→ click sector → IMAP→ click peer → RELS→ click ticker → DES→ click field → Inspector

There is no limit to how deep you can drill. Every view offers next steps. The back button (Ctrl+B) always takes you one step back.

10. Overlays: Inspector, Menu, Search

HL Search Overlay (Ctrl+K or type HL)

The most powerful discovery tool. Opens a full-panel search over all securities and all functions simultaneously.

  • Type any part of a ticker, company name, or function code
  • Results show fn for function or → for security, with full name and category
  • Navigate with ↑↓, confirm with Enter
  • Selecting a function runs it on the current security
  • Selecting a security loads it and keeps the current function
  • Close with Esc

MENU Overlay (F2)

Shows contextual next steps based on the currently loaded security and its sector.

  • Two tabs: Security Functions (sector-aware) and Global Functions
  • Equity panel: shows DES, HP, GP, FA, CN, OWN, RELS, MGMT, DVD, EVT, ORD, ALRT, TOP, WEI, IMAP
  • FX panel: shows DES, GP, FXC, ECO
  • Click any item or type its number and Enter

Inspector Overlay (Alt+Click or Alt+Enter)

Opens a dense side-panel (right edge) without destroying your current view. Shows all known fields for any entity.

  • Header shows entity kind, name, and timestamp
  • Body: key-value grid of all fields — every cyan field is itself clickable
  • Footer: Related functions list + Open/Send-to-Panel buttons
  • Pin button — keeps inspector open as you click other entities
  • Send to Panel — drills the inspected entity in the next panel
  • Close with ✕ or Esc (unless pinned)

Context Menu (Right-Click)

Available on any entity — row in a table, headline in news, field in DES, chart element.

12. Alerts Engine

Set price triggers on any security. When triggered, the global status bar turns red with a badge count and the ALRT panel highlights the triggered rule.

Creating Alerts

ALERT IF SPY > 510Alert when SPY crosses above $510
ALERT IF AAPL < 170Alert when AAPL drops below $170
ALRTOpen the alerts monitor panel to manage all rules

From Command Line:

Type the rule exactly as shown above in any panel's command line and press Enter.

Format: ALERT IF [SYMBOL] [>|<] [PRICE]

From Context Menu:

Right-click on any numeric field (P/E, price, yield, etc.) in DES, HP, OWN, or any table — select "Alert on field" to create a pre-filled alert.

Alert rules are saved to local browser storage and persist across sessions.

13. Watchlists (MON)

MON is your personal live monitor — a real-time table of any securities you add. Data updates automatically from the B-PIPE simulation.

Accessing MON

  • Type MON + Enter in any panel
  • Click the MON button in the global status bar

Managing the Watchlist

Type symbol in input boxe.g. "TSLA US Equity" or just "TSLA" — press Enter or ADD button
✕ button on any rowRemove that symbol from watchlist
Click any rowDrill into that security (DES by default)
Shift+Click any rowOpen that security in a new panel
Right-click any rowFull context menu with DES/GP/CN/etc.

What MON Shows

Columns: Security name, Last price, % Change (color coded), Volume (M), VWAP (volume-weighted average price from worker analytics).

Add from Context Menu
Right-click on any ticker in any table → "★ Add to monitor" — instantly adds it to your MON watchlist without leaving your current view.

15. Panel Sync (Link Groups)

Link groups let you connect panels so that changing the security in one panel automatically updates others in the same group.

Setting Up a Link Group

1
In a panel header, click the small colored square (■) in the top-right corner
2
Each click cycles through: no link (dark) → Red → Green → Blue → Yellow → back to no link
3
Set two panels to the same color (e.g., both Red)
4
Now change a security in one of the Red panels — the other Red panel automatically loads the same security
Classic Bloomberg Workflow
Set Panel 1 (WEI) and Panel 2 (DES) both to "Red". Browse the WEI table — click any index. Panel 2 (DES) instantly updates to show details for whatever you're looking at. Panel 3 (GP, also Red) updates too.
Note
Link groups sync the security only — not the function. Panel 2 keeps showing DES even when Panel 1 switches to a different function.

19. Troubleshooting & Tips

Simulation Mode
All data in MarketMind Terminal is simulated. Prices, news, ownership data, and analytics are generated algorithmically and do not represent real market data. This is clearly indicated by the SIM badge visible throughout the interface.

Common Situations

SituationSolution
Command doesn't load anything Make sure you pressed Enter. Autocomplete showing doesn't execute — you must confirm.
Can't type in the command line Click the panel first to focus it, then press Ctrl+L to focus the command line.
Overlay is stuck open Press Esc to close any open overlay (inspector, menu, search, help).
Panel shows a "WAKE UP" screen That's intentional — it means no function is loaded. Type any command or click an example to start.
Back button (◀) is greyed out You're at the beginning of this panel's history — nothing to go back to.

Power User Tips

  • The fastest way to research anything is: Ctrl+K → type → Enter. No mouse needed.
  • Use the Next Actions strip (the row of small amber buttons under the breadcrumb) to navigate between related functions instantly with one click.
  • Use Shift+Click constantly to fill multiple panels with related data without disrupting what you're already looking at.
  • The Inspector overlay (Alt+Click) is perfect for quickly checking field values without losing your scroll position in a table.
  • Right-click anywhere you see data — the context menu always has at least 3-5 relevant actions.
  • FA (Financial Analysis) has three tabs at the top: Income Statement, Balance Sheet, Cash Flow. Click to switch.
  • In GP chart, type a ticker in the "vs…" box to compare two securities on the same chart.
  • The breadcrumb (e.g. EQUITY › AAPL US › DES) shows exactly where you are and can remind you of what path you took.
