# PREF — Preferences & Settings

<span class="mnemonic-badge">PREF</span>
<span class="scope-badge">🌍 Global</span>
<span class="scope-badge">OpsConsole</span>

## Purpose

User settings panel: display density, layout mode, streaming mode, and advanced settings links.

## Inputs

No security required.

## Screen Layout

1. **Density picker (Comfortable / Default / Compact)**
2. **Layout mode (Tile / Tab / Stack)**
3. **Live Mode / High Density toggles**
4. **Advanced settings quick-links**

## Key Fields

_No specific fields — navigational/administrative function._

## Drill Paths

- Click advanced links → KEYMAP, LAYOUT, FLD, POLICY, ENT, AUD, API, SRC

## Keyboard Controls

- Density changes persist to localStorage across sessions

## Opening Examples

```
PREF GO
```

## Common Pitfalls

- ⚠️ Density changes update token values but require reload to fully propagate to all components.
