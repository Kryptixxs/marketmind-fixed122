# ECO — Economic Calendar

<span class="scope-badge">🌍 Global</span>

## Purpose

Upcoming macro releases with country, importance, consensus, prior, and actual (where released).

## Inputs

No security. Timeframe optional.

## Screen Layout

1. **Calendar table (date, event, country, importance, consensus, prior, actual)**
2. **Surprise index strip**
