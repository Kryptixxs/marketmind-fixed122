# NAVTREE — Global Function Navigator

<span class="mnemonic-badge">NAVTREE</span>
<span class="scope-badge">🌍 Global</span>
<span class="scope-badge">OpsConsole</span>

## Purpose

Browse, filter, and launch all 2,949+ functions in the mnemonic catalog. Supports category filters, taxonomy groups, favorites, and recent lists.

## Inputs

No security required.

## Screen Layout

1. **Full catalog table with search**
2. **Category filter dropdown**
3. **Favorites/Recent/Pinned filters**
4. **Taxonomy group count badge**

## Key Fields

| Field ID | Description |
|---|---|
| `CODE` | SIM-sourced CODE value |
| `TITLE` | SIM-sourced TITLE value |
| `TAXONOMY` | SIM-sourced TAXONOMY value |
| `RELATED` | SIM-sourced RELATED value |

## Drill Paths

- Click any row → launch that function in focused pane

## Keyboard Controls

- Type to filter → instant results
- Category dropdown → filter by asset class
- Favorites/Recent/Pinned filter tabs

## Opening Examples

```
NAVTREE GO
```

## Common Pitfalls

- ⚠️ Shows up to 1800 results. Use search to narrow down.
