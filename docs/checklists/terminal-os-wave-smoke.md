# Terminal OS Wave Smoke Checklist

## Command Line + Audit
- Enter `AAPL DES GO`, `AAPL HP GO`, `AAPL NOTES GO` and confirm panel navigation.
- Enter `WS TESTA GO` (save), change layouts, then `WS TESTA GO` (load).
- Enter `WS DEL TESTA GO` and verify workspace deletion.
- Run `GRAB GO` and confirm export tab opens.
- Open `AUD GO` and verify new entries for GO/workspace/export/drill/alerts.

## Universal Drill Contract
- In `MON`, `BLTR`, `DES`, and news rows:
  - Click opens in place
  - Shift+Click sends to another panel
  - Alt+Click opens inspector
  - Right-click opens context menu
- Verify keyboard row drill:
  - Arrow up/down changes selection
  - Enter drills
  - Shift+Enter sends
  - Alt+Enter inspects

## Workspace Recovery
- Interact for 30+ seconds and refresh the app.
- Confirm panel mnemonic/security/history/timeframe/selection restore from recovery snapshot.
- Confirm focused panel restore and command histories persist.

## MON + BLTR
- In `MON`, create a new named list, add symbols, filter, sort, and export.
- In `BLTR`, toggle status filters and confirm rows update.

## IB + NOTES
- In `IB`, send a message with context attachment on and off.
- In `DES`, add/delete notes and confirm they persist for the same security.
- In `NOTES`, verify per-security note list and delete flow.
